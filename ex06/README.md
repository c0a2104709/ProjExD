## 第3回迷路ゲームに追加：（ex06/Pack-Man_Kokaton.py）
### ゲーム概要
- ex06/Pac-Man_Kokaton.pyを実行すると，1500x720のcanvasにステージが描画され，敵を避けながら、60秒間迷路に沿ってこうかとんを移動させるゲーム

- 矢印キーでこうかとんを上下左右に移動する
